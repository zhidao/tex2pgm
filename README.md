# tex2pgm ver2.7
                                                by Zhidao
                                                2003.12.30. 作成
                                                2010. 8.31. 最終更新
--------------------------------------------------------------------
### 【説明】

TeXの書式で書かれたファイルをEPSファイルおよびPGMファイルに変換する
スクリプト。実行には LaTeX、dvips、pstopnm が必要です。

--------------------------------------------------------------------
### 【使い方】

例えば eqn.tex をEPSファイルに変換したければ、

% tex2pgm eqn.tex

あるいは

% tex2pgm eqn

でファイル eqn.eps および eqn.pgm が出力されます。
TeXファイルの書式は sample.tex を参考にして下さい。

% tex2pgm eqn1 eqn2 eqn3 ...

でeqn1.tex, eqn2.tex, eqn3.tex, ... と次々に変換していきます。

% tex2pgm -p preamble.tex eqn1 eqn2 eqn3 ...

で、preamble.texを先に読み込むことが可能です。

% tex2pgm -u package.sty eqn1 eqn2 eqn3 ...

で、package.styを使用することができます。
プリアンブル、パッケージとも幾つでも指定できますが、必ず数式を書いた
ファイルよりも先に指定して下さい。また、拡張子は全て省略できます。

% tex2pgm -u package.sty eqn*

のように、ワイルドカードも使えます。

% tex2pgm -e "\\alpha x + \\beta"

などとすれば、コマンドライン引数に直接TeXの書式を与えることもできます。

% tex2pgm -e "\bm{x}"

のように、太字イタリックを出力する\bmコマンドがデフォルトで定義されて
います。
